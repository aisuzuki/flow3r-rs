# Rust support for flow3r-rs

https://flow3r.garden/

## Goals
- Rust implementation of Flow3r board

## Setup development environment

1. Install all requirement including toolchains described in [here](https://docs.espressif.com/projects/rust/book/getting-started/toolchain.html#toolchain-installation).
2. As described in [Environment Variables Setup](https://github.com/esp-rs/espup?tab=readme-ov-file#environment-variables-setup), source the environment variables:
   ```sh
   source $HOME/export-esp.sh
   ```
